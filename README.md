# Workout Planner AI MCP

Plan workouts, calculate BMI and one-rep max estimates, and suggest exercise splits.

## Tools

- `calculate_bmi` — Calculate BMI and category.
- `calculate_one_rep_max` — Estimate one-rep max using common formulas.
- `suggest_workout_split` — Suggest a weekly workout split based on available days.

## Install

```bash
pip install -e .
```

## Run

```bash
python -m workout-planner-ai-mcp.server
```
